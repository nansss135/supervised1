# House Price Prediction using Regression Algorithms

## Project Overview

This project demonstrates the implementation of regression algorithms to predict house prices using machine learning. Different regression techniques were applied and compared to understand their performance on a house price dataset.

The project also explores model evaluation metrics, polynomial regression, gradient descent optimization methods, and the bias-variance trade-off.

---

## Dataset Features

* House_Area_sqft
* Bedrooms
* Bathrooms
* Location_Score
* Age_of_Property
* House_Price (Target Variable)

---

## Algorithms Implemented

### Simple Linear Regression

* Single independent variable
* House Area used to predict House Price

### Multiple Linear Regression

* Uses all independent variables
* Improved prediction accuracy

### Polynomial Regression

* Degree 2
* Degree 3
* Comparison with Linear Regression

## Conclusion

The project compared different regression algorithms for house price prediction. Multiple Linear Regression generally provided better performance than Simple Linear Regression because it utilized multiple features. Polynomial Regression was effective in modeling non-linear relationships but required careful selection of the polynomial degree to avoid overfitting.
